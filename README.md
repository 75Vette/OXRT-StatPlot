# OXRT-StatPlot
Overview
--------
OXRT-StatPlot is a portable, zero-install visualisation tool for analysing performance logs generated by the OpenXR Toolkit. It reads .csv performance logs and produces high-quality neon-themed PNG plots showing:

- FPS over time
- CPU App, CPU Render, and GPU frametimes
- Frametime distribution (0–20 ms)
- Overlay or split-view modes
- Automatic refresh-rate detection
- Optional smoothing
- Automatic output directory creation
- Automatic config file generation

The tool is fully interactive and requires no installation. Just run the executable.

Portable Usage
--------------
1. Download the ZIP from GitHub.
2. Extract anywhere.
3. Run:

    OXRT-StatPlot-0.9.0.exe

The tool will automatically create its configuration file and output directory on first run.

CSV Log Directory Behaviour
---------------------------
On startup, OXRT-StatPlot attempts to locate the default OpenXR Toolkit stats folder:

    %USERPROFILE%\AppData\Local\OpenXR-Toolkit\stats

If this folder exists, the tool automatically switches into it.

If it does not exist, the tool will:

1. Prompt for the correct path.
2. Save that path into config.ini.
3. Use it automatically on future runs.

Output Directory
----------------
All generated PNG files are saved into:

    OXRT StatPlot Results\

This folder is created next to the executable.

Example output filename:
	
	OXRT StatPlot dd-Mmm-yy - hhmm - <user entered title>.png
    OXRT StatPlot 25-Apr-26 - 1842 - My Title.png

Config File (config.ini)
------------------------
A config.ini file is created next to the executable on first run.

Example:

    [paths]
    stats_dir = auto
    output_dir = auto

    [defaults]
    resolution = 4k
    smoothing = medium
    refresh_rate = auto
    plot_mode = split
    title_mode = default

    [ui]
    show_ascii_art = yes
    auto_open_png = yes
    verbose = no

Key Behaviours
--------------
stats_dir = auto  
    Uses the OpenXR Toolkit default stats folder.

output_dir = auto  
    Creates "OXRT StatPlot Results" next to the executable.

show_ascii_art = yes/no  
    Toggles the startup ASCII banner.

auto_open_png = yes/no  
    Opens PNGs automatically after saving.

title_mode = default/manual/none  
    Controls plot title behaviour.

resolution  
    4k, 1440p, or 1080p.

smoothing  
    none, light, medium, or heavy.

Delete config.ini at any time to restore factory defaults.

Features
--------
FPS Split Mode  
    One subplot per CSV. FPS + CPU App + CPU Render + GPU. Neon colour palette. Automatic refresh-rate scaling.

FPS Overlay Mode  
    All FPS lines on one plot. Automatic fallback to 50–150 Hz if refresh rates differ.

Frametime Distribution (Split)  
    GPU + CPU App + CPU Render. 0–20 ms. 0.1 ms bins. Vertical Hz lines.

Frametime Distribution (Overlay)  
    GPU + CPU App. Colour-paired overlays. Vertical Hz lines.

Smoothing Options  
    None, Light (8 samples), Medium (20 samples), Heavy (40 samples).

Resolution Options  
    4K (3840×2160), 1440p, 1080p.

PNG Naming Format  
    OXRT StatPlot dd-Mmm-yy - hhmm - {title}.png
